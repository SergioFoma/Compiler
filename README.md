# Установка программы 👾

### **Чтобы скачать весь проект, нужно:**

1. В Вашем терминале необходимо указать директорию, куда вы хотите скачать данный проект.

2. Затем в терминале введите.
```
git clone git@github.com:SergioFoma/Compiler.git
```

3. Дождитесь скачивания.

4. После чего будет создана рабочая директория с названием репозитория.

# Запуск программы 🚀

Чтобы запустить программу на языке ШЛЯПНЫЙ ПАБГ, необходимо:

1. Открыть текстовый документ ```code.txt``` в папке ```Compiler/frontend```.

2. Написать в данном файле свой код на языке ```шляпный пабг```

>⚠️ Проект использует фиксированную структуру каталогов для обеспечения работы конвейера сборки.
> Скрипт compile.sh жестко связывает этапы (Frontend, Middleend, Backend), гарантируя передачу промежуточных артефактов (AST, ассемблерный код) между модулями.
> При необходимости пути легко переопределяются через переменные внутри скрипта.

3. Прописать в консоли команду.
```
commonFiles/compile.sh
```

Данная команда запустит компиляцию всех файлов проекта и начнёт его исполнение.


4. Результатом выполнения программы является байт-код, готовый к исполнению.

# Архитектура компилятора ⚙️
Компилятор написан на C/C++ и состоит из трёх частей:

1. Frontend: Лексический и синтаксический анализ. Переводит исходный код(```code.txt```) в абстрактное синтаксическое дерево (AST). Структуры узлов AST реализованы с помощью union, что позволяет хранить разные типы данных(числа, переменные, операторы).

2. Midleend: Здесь происходит оптимизация AST - свертка констант(умножени на 0, сложение двух чисел, умножение на 1 и тд...)

3. Backend: Генерация ассмблерного кода. Обходит AST и генерирует ассемблерный код(код генерируется для ранее написанного мной процессора SPU)

4. ASM: Генерация байт кода. Осуществляется перевод ассемблерыных команд в байт-код.

>⚠️ Определение функции начинается с ключевого слова  IN_PUBG </div>

```Пример определения функции```

```c
IN_PUBG factorial( number_for_factorial ){

    var_in_factorial pabgo_govnito 0 gusi

    wool( number_for_factorial == 1 ){
        var_in_factorial pabgo_govnito 1 gusi

}

```
